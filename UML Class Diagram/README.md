UML Class Diagram — Каршеринг
Учебный проект с UML-диаграммой классов сервиса каршеринга.
  Описание
Диаграмма отражает базовую объектную модель системы аренды авто:
  управление пользователями
  аренда автомобилей
  обработка платежей
  статусы машин и поездок

Основные классы
Пользователь — профиль и способы оплаты
Автомобиль — данные авто и его состояние
Аренда — жизненный цикл поездки (главный класс)
Оплата — проведение платежа и возвраты
